ABOUT

SAZI NANASHIKI

秋本すばこ

→ MORE INFO

幼い頃からピアノに親しみ、2007年から楽曲制作を始める。
2010年に個人サークル（旧名：STUDiO MONOBEAT）を立ち上げ、初期はインストゥルメンタルの楽曲を中心に活動を行う。
現在はVOCALOIDなどの音声合成含むボーカリストを招き、作編曲・作詞からデザインを単独で担当する。
主な制作環境として、Apple Logic ProとPresonus Studio Oneを使用。
